# 🚀 PQ Engine (v0.7 Beta)
> **"PHP의 복잡함을 걷어 내고 쉽고 편리안 엔진을 장착한다."**


**PQ(PHP+QUERY / PHP+QUICK)** 엔진은 
"PHP+QUERY / PHP+QUICK" 이 두가지 의미를 축약해서 [PQ] 라 하게 되었습니다.

PHP의 복잡함을 걷어내고, 직관성을 유지하는데 촛점을 맞춥니다.
변수는 @
객체는 (@)
배열은 ([])
체이닝 .
[[pq구문]] 으로 변경진행중...

👉pq는 php v8 기반 위에서 돌아가는 엔진입니다.
